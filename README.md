# Cheat sheet.

> If you want to master something, teach it.
> —  Richard Feynman
>
> 學習技巧有很多，但我覺得有個方法特別重要，就是把自己所學的說給別人聽，源自於[費曼學習法](https://wiki.mbalib.com/zh-tw/%E8%B4%B9%E6%9B%BC%E5%AD%A6%E4%B9%A0%E6%B3%95)，方法很簡單卻很有效果，從別人的反饋中我們可以了解到自己對該知識掌握薄弱的點。很多時候大腦在學習的過程中會有很多模糊不清的點，如果不說出來的話，這些不清楚地點會被忽略掉，但如果要讓別人聽得懂，那我們需要懂的更多才行。
> 整理筆記、寫文章其實也是說給別人聽的一種方式，只不過比單純的說要更為系統。所以我一般在學習某個技術的時候會去寫文章記錄下來。一方面讓自己的知識梳理的更清晰，一方面也是幫助以後的自己忘記時可以回來查看，不至於白白浪費了之前學習的時間。
>
