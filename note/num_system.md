# Numeral system (數字系統/記數系統)

## 進制簡介
- 十進制 (Decimal, dec)，以 『10』 做為進位基底 (base) 的數字系統，逢十進位，因此又稱為 base-ten、base 10。
- 二進制 (Binary)，又稱 base 2
- 八進制 (Octal, oct) 
- 十六進制 (Hexadecimal, hex)

```
>>> '\u4e2d\u6587'
'中文'
```
\u4e2d代表是使用unicode表示的字符
